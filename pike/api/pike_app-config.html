class Pike\AppConfig

Varastoi App->__construct():lle tarjotun konfiguraation.

Synopsis 

class AppConfig {

    /* Metodit */
    public get ( string $key, mixed $default = null ) : mixed|null
    public getVals ( void ) : object
    public setVals ( object|array<string, mixed> $config ) : void
}

Metodit

get()

Palauttaa konfiguraatioarvon tai $default, mikäli sellaista ei löytynyt.

Signature 

public function get ( string $key, mixed $default = null ) : mixed|null

Esimerkit 

$single = $config->get('key', 'none');

getVals()

Palauttaa konfiguraatiot kokonaisuudessaan.

Signature 

public getVals ( void ) : object

Esimerkit 

$all = $config->getVals();

setVals()

Yliajaa konfiguraatiot kokonaisuudessaan.

Signature 

public function setVals ( object|array<string, mixed> $config ) : void

Esimerkit 

$config->setVals((object) ['foo' => 'bar']);
