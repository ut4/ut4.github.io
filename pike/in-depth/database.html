Tietokanta

Pikessä tietokantakyselyt suoritetaan \Pike\Db-luokalla, joka on on ohut wräpperi \PDO-abstraktion ympärille.

use \Pike\Db;
class MyCtrl {
    public function __construct(Db $db) {
        $this->db = $db;
    }
    // tai..
    public function myMethod(Db $db) {
        // tee jotain $db:llä
    }
}

Käyttöönotto (MariaDb/MySQL)

\Pike\Db:n konfigrointiin tarvitaan kolme asiaa:

  • \Pike\App::create():een passattu $config, jossa vähintään 'db.host' => 'myval'
  • \Pike\App::create():een passattu $ctx, jonka konstruktoriin ['db' => '@auto']
  • $db->open()-kutsu

Näiden jälkeen \Pike\Db injektoituu minkä tahansa kontrollerin konstruktoriin tai metodiin type-hinttien perusteella (kuten esim. Pike 101 -esimerkin kontrolleri). \Pike\Db -luokasta luodaan vain yksi instanssi, ks. \Auryn\Injector->share().

Esimerkki:

index.php 

$config = [
    'db.host'        => '127.0.0.1', // oletus '127.0.0.1'
    'db.database'    => 'new2',      // oletus ''
    'db.user'        => 'devuser',   // oletus ''
    'db.pass'        => 'qweqwe',    // oletus ''
    'db.tablePrefix' => 'rad_',      // oletus ''
    'db.charset'     => 'utf8mb4',   // oletus 'utf8mb4'
];
// tai
$config = __DIR__ . '/config.php'; // jossa <?php return [...];

$serviceHints = ['db' => \Pike\App::MAKE_AUTOMATICALLY];
// tai
$serviceHints = ['db' => '@auto'];

$app = \Pike\App::create([MyBootstrapModule::class],
                         $config,
                         new \Pike\AppContext($serviceHints));
$app->handleRequest(...);

MyBootstrapModule.php 

abstract class MyBootstrapModule {
    /**
     * @param \Pike\AppContext $ctx
     */
    public static function init(\Pike\AppContext $ctx): void {
        try {
            $ctx->db->open();
        } catch (\Pike\PikeException $e) {
            // Tee jotain
        }
    }
}

Datan insertointi, yksi rivi 

$data = (object) ['foo' => 'value 1', 'bar' => 'value 2'];
[$qList, $values, $columns] = $db->makeInsertQParts($data);
try {
    $numAffectedRows = $db->exec("INSERT INTO `Products` ({$columns}) VALUES ({$qList})",
                                 $values);
    if ($numAffectedRows > 0)
        echo $this->db->lastInsertId(); // '14'
    else
        ; // Tee jotain
} catch (\Pike\PikeException $e) {
    // Handlaa tilanne
}

Vain valitut kentät 

$data = (object) ['foo' => 'value 1', 'bar' => 'value 2', 'baz' => 'value 3'];
$onlyTheseColumns = ['foo', 'baz'];
[$qList, $values, $columns] = $db->makeInsertQParts($data, $onlyTheseColumns);
//
$qList;   // '?,?'
$columns; // '`foo`,`baz`'
//
$numAffectedRows = $db->exec("INSERT INTO `Products` ({$columns}) VALUES ({$qList})",
                             $values);

Datan insertointi, useita rivejä 

$data = [
    (object) ['foo' => 'value1', 'bar' => 'another value'],
    (object) ['foo' => 'value2', 'bar' => 'another value2'],
];
[$qGroups, $values, $columns] = $db->makeBatchInsertQParts($data);
try {
    $numAffectedRows = $db->exec("INSERT INTO `Products` ({$columns}) VALUES {$qGroups}",
                                 $values);
    if ($numAffectedRows > 0)
        echo $this->db->lastInsertId(); // '14'
    else
        ; // Tee jotain
} catch (\Pike\PikeException $e) {
    // Handlaa tilanne
}

Vain valitut kentät 

$data = [(object) ['foo' => 'value 1', 'bar' => 'value 2', 'baz' => 'value 3'],
         (object) ['foo' => 'value 4', 'bar' => 'value 5', 'baz' => 'value 6']];
$onlyTheseColumns = ['bar', 'baz'];
[$qGroups, $values, $columns] = $db->makeBatchInsertQParts($data, $onlyTheseColumns);
//
$qGroups; // '(?,?),(?,?)'
$columns; // '`bar`,`baz`'
//
$numAffectedRows = $db->exec("INSERT INTO `Products` ({$columns}) VALUES {$qGroups}",
                             $values);

Datan hakeminen, yksi rivi 

try {
    $row = $db->fetchOne("SELECT `foo`,`bar` FROM Products WHERE `id`=?", [1]);
    if ($row)
        echo $row['foo']; // 'value'
    else
        ; // $row === null|false, tee jotain
} catch (\Pike\PikeException $e) {
    // Handlaa tilanne
}

Datan päivittäminen 

$data = (object) ['foo' => 'value 1', 'bar' => 'value 2'];

try {
    $numAffectedRows = $db->exec("UPDATE `Products` SET {$columns} WHERE `id`=?",
                                 array_merge($values, [1]));
    if ($numAffectedRows > 0)
        ; // ok
    else
        ; // Tee jotain
} catch (\Pike\PikeException $e) {
    // Handlaa tilanne
}
